(1) (2x + 6)/5 = x/2
x = 12
Multiply both sides by 10 (LCM=10): 2·(2x+6) = 5x → 4x + 12 = 5x → x = 12.
(2) (3x + 9)/3 = x/7
x = -7/2
Multiply both sides by **21** (a common multiple, not lowest): 7(3x+9) = 3x 21x + 63 = 3x 18x = -63 → x = -63/18 = -7/2.
(3) (5x + 2)/5 = x/4
x = -8/15

(4) (3x + 7)/2 = x/3

(6) (7x + 9)/5 = (3x + 4)/2 + x/10
