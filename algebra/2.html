Highest Common Factor (HCF) – Linear and Quadratic Forms

Key Idea: For every quadratic (and linear) expression you meet in this chapter, the first thing to check is whether there is a Highest Common Factor (HCF). If all terms are even, or all are multiples of the same number (and sometimes the same power of \(x\)), you should factor that number (and \(x\)) out before doing anything else.

This simple habit will make all later work with quadratics easier – factoring, completing the square, using the formula, and sketching graphs.

Multiply the factor outside the bracket by each term inside:

Example (d):

Notice the pattern: each expression ends up as either \(ax + bx\) or \(ax^2 + bx\).

2. Factorising by HCF: From \(ax + bx\) and \(ax^2 + bx\)

Now work backwards. Take expressions of the form \(ax + bx\) and \(ax^2 + bx\), and factor out the highest common factor. This is the first step in almost every quadratic you will meet later.

Classwork – Factor out the HCF.

(a) \(15x\)
(b) \(20x\)
(c) \(15x\)
(d) \(4x^2 + 6x\)
(e) \(9x^2 - 3x\)
(f) \(-12x^2 + 18x\)
(a) \(15x = 3x(5)\)
(b) \(20x = 4x(5)\)
(c) \(15x = 5x(3)\)
(d) \(4x^2 + 6x = 2x(2x + 3)\)
(e) \(9x^2 - 3x = 3x(3x - 1)\)
(f) \(-12x^2 + 18x = -6x(2x - 3)\)
Look for the biggest number and highest power of \(x\) that divides every term.
Example (d): Both terms have a factor \(2x\):
\(4x^2 = 2x \cdot 2x,\quad 6x = 2x \cdot 3\).
So \(4x^2 + 6x = 2x(2x + 3)\).
Homework – Factor out the HCF.

(a) \(21x\)
(b) \(35x\)
(c) \(10x\)
(d) \(4x^2 + 5x\)
(e) \(6x^2 - 12x\)
(f) \(-12x^2 - 4x\)
(a) \(21x = 7x(3)\)
(b) \(35x = 5x(7)\)
(c) \(10x = 2x(5)\)
(d) \(4x^2 + 5x = x(4x + 5)\)
(e) \(6x^2 - 12x = 6x(x - 2)\)
(f) \(-12x^2 - 4x = -4x(3x + 1)\)
Try to write each term as (HCF) × (something). The bracket collects all those “somethings”.
Revision – Factor out the HCF.

(a) \(18x^2 + 45x\)
(b) \(-12x^2 + 21x\)
(c) \(24x^2 + 16x\)
(d) \(-10x^2 - 45x\)
(a) \(18x^2 + 45x = 9x(2x + 5)\)
(b) \(-12x^2 + 21x = 3x(-4x + 7)\)
(c) \(24x^2 + 16x = 8x(3x + 2)\)
(d) \(-10x^2 - 45x = -5x(2x + 9)\)

3. Setting Equal to Zero: Factors, Roots and Simple Graphs

When an expression equals zero, your factors become very powerful. You can find the roots (solutions) and understand what the graph looks like.

Classwork – Solve using HCF and describe the graph.

(a) \(4x^2 + 6x = 0\)
(b) \(6x^2 - 12x = 0\)
(a) Roots: \(x = 0\) and \(x = -\dfrac{3}{2}\).
(b) Roots: \(x = 0\) and \(x = 2\).
(a) \(4x^2 + 6x = 0 \Rightarrow 2x(2x + 3) = 0\).
So \(2x = 0\) or \(2x + 3 = 0\).
\(x = 0\) or \(x = -\dfrac{3}{2}\).

(b) \(6x^2 - 12x = 0 \Rightarrow 6x(x - 2) = 0\).
So \(6x = 0\) or \(x - 2 = 0\).
\(x = 0\) or \(x = 2\).
Both are parabolas (quadratic graphs).
(a) Crosses the x-axis at \(0\) and \(-1.5\). Opens upwards.
(b) Crosses the x-axis at \(0\) and \(2\). Opens upwards. The y-axis is at \(x = 0\), so one root is on the y-axis.
Homework – Solve using HCF.

(a) \(9x^2 - 3x = 0\)
(b) \(-12x^2 + 18x = 0\)
(a) Roots: \(x = 0\) and \(x = \dfrac{1}{3}\).
(b) Roots: \(x = 0\) and \(x = \dfrac{3}{2}\).
Factor out the HCF first, then use “factor = 0” on each bracket.
Revision – Mix of linear and quadratic.

(a) \(24x^2 + 16x = 0\)
(b) \(-10x^2 - 45x = 0\)
(c) \(21x = 0\)
(a) \(8x(3x + 2) = 0 \Rightarrow x = 0\) or \(x = -\dfrac{2}{3}\).
(b) \(-5x(2x + 9) = 0 \Rightarrow x = 0\) or \(x = -\dfrac{9}{2}\).
(c) \(21x = 0 \Rightarrow x = 0\) (a straight line through the origin).
Reflection:
In every example on this page, the first step was to look for a Highest Common Factor (HCF).

• For expressions like \(ax + bx\) and \(ax^2 + bx\), factoring out the HCF changes them into a factor times a bracket.
• For equations like \(ax^2 + bx = 0\), factoring out the HCF lets us find the roots quickly and sketch simple graphs.

This HCF step will appear again and again in all later work with quadratics. Always look for it first.
