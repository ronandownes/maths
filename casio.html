CASIO Calculator Skills

ClassWiz and similar models – used properly, not as a crutch.

These pages will be embedded into lessons (e.g. Functions, Quadratics, Statistics) so you can see exactly how to use the calculator for the learning outcomes in each topic.

1. Getting Started

Here we will cover basic setup: modes, clearing, fixing mistakes, and checking that your calculator is in the correct setting for exams.

2. Equations and Quadratics

This section will show how to use the equation solver for linear and quadratic equations, and when you should not rely on it (for example, when you still need to show algebraic work for full marks).

3. Statistics Mode

Using the STAT mode for mean, standard deviation, regression lines, and checking answers quickly during revision.

4. Tables, Functions and Graph Thinking

Using the TABLE mode to explore functions such as $f(x)=2x^2-3x+5$, spot patterns, and connect calculator output back to the algebra and graphs you see in class.

5. Logs, Indices and Other Features

Later we can add worked steps for $\log$, powers, absolute values, and the SOLVE feature, aligned with the Number and Algebra learning outcomes.
